Help Desk

A web application made from AdonisJS framework which aims to help developers and administrators keep up with the client's technical issues.

The highlight of the web application is it's real-time messaging, real-time ticket updates and notifications. Redis and socket.io were some of the plugin used for the live chat and real-time notifications.

Auto Dialer

This web application is also made from AdonisJS framework and it was mainly used as the in-house call center software for the company. The company has it's own PBX, asterisk.io and express.js were used to create a service/listener that will handle the requests from the website to PBX.

Researchers can also upload leads and the application has an algorithm that will filter duplicates and other criteria.
